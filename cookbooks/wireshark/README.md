Description
===========

Installs Wireshark 1.8.6 Intel 64

Requirements
============

## Platform:

* Mac OS X, 10.8 or higher

## Cookbooks:

* dmg, v1.1.0+
* xquartz, v1.0.0+

Attributes
==========

* `node['wireshark']['version']` - Version to install.
* `node['wireshark']['url']` - Download URL.
* `node['wireshark']['checksum']` - SHA256 checksum of downloaded dmg.
